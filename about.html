Sobre o Site

Este projeto é um site simples feito com HTML, CSS e JavaScript. Ele tem quatro páginas principais: uma inicial (index.html), uma de formulário (form.html),
uma para exibir os dados do formulário (formAction.html) e uma sobre o projeto (about.html).

O site usa CSS para o visual e JavaScript para validações e interatividade. Os arquivos estão organizados em pastas para facilitar o entendimento.
É um exemplo básico de como criar e organizar um site funcional.
